## ğŸš€ Instalasi & Penggunaan

1.  **Clone Repository**
    ```bash
    git clone https://github.com/azmil72/hmti.git
    cd hmti
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Jalankan Development Server**
    ```bash
    npm start
    ```
    Aplikasi akan berjalan di http://localhost:3000.

4.  **Build untuk Produksi**
    ```bash
    npm run build
    ```
